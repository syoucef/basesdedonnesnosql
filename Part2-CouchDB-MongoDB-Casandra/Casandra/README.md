# Système de gestion de bases de données NoSQL Casandra

Comme pour les deux systèmes NoSQL CouchDB et MongoDB, l'objectif dans ce qui suit est de vous introduire Casandra. Pour répondre aux problamétiques de stockage et de gestion de très gros volumes de données à grande échelle, un groupe d'ingénieur de Facebook (2007) ont proposé Casandra. Avant qu'elle ne soit portée par Incubator, en 2010, Casandra n'a pas eu de succès. Depuis, elle est elle promu au rang de top-level Apache Project.
